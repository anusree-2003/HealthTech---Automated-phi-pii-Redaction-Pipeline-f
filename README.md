## ▶️ How to Run

Clone the repository:

```bash
git clone https://github.com/anusree-2003/HealthTech---Automated-phi-pii-Redaction-Pipeline-f.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
uvicorn api.main:app --reload
```

Open:

```
http://127.0.0.1:8000
```
